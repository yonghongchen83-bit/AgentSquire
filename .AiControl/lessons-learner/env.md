# Lessons Learner Environment

## Purpose
Formalize the process of documenting lessons learned after each development session. Ensures knowledge is captured, structured, and discoverable by future agents.

## Directory
- `lessons-learner/skill.md` — This skill's workflow
- `lessons-learned/lessons.md` — Index of all lessons
- `lessons-learned/NNN-slug.md` — Individual lesson detail pages
- `root/state.md` — Root-level reference table
- `<node>/state.md` — Per-node links to relevant lessons

## Tools Used
- `Read` — to review conversation history and existing docs
- `Write` — to create lesson pages
- `Edit` — to update existing index, state, env, and skill files
