# Prompt

Use this node to finalize architecture decisions for a per-session ContextManagerAdapter that supports:

1. Legacy mode: full conversation history replay.
2. Squire mode: dynamic curated context via Squire protocol.

Deliverables for this node:

- Adapter contract and responsibility boundaries.
- Session-level switching strategy.
- Storage and migration plan.
- Failure and fallback behavior.
- Test strategy.
- Resolved answers for all open ambiguities before coding.
